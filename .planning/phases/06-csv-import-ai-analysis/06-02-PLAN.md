Create `src/lib/anthropicClient.ts` as a pure TypeScript module (no React imports) with the following exports:

**Types:**
```typescript
export interface BucketSuggestion {
  spendingAverageEur: number;
  monthsAnalyzed: number;
  reasoning: string;
  suggestedMonthlyAmountEur: number;
}

export interface AIAnalysisResult {
  everydayEssentials: BucketSuggestion;
  funDiscretionary: BucketSuggestion;
  oneOffTravel: BucketSuggestion;
  recurringFixed: BucketSuggestion;
}

export class AnthropicAPIError extends Error {
  constructor(public readonly status: number, public readonly errorType: string) {
    super(`Anthropic API error ${status}: ${errorType}`);
    this.name = 'AnthropicAPIError';
  }
}
```

**JSON Schema constant** (use the exact schema from 06-RESEARCH.md Code Examples section — the ANALYSIS_SCHEMA object with all 4 bucket objects, each requiring spendingAverageEur, monthsAnalyzed, reasoning, suggestedMonthlyAmountEur, additionalProperties: false).

**Prompt builder:**
```typescript
function buildAnalysisPrompt(transactions: import('@/lib/csvParser').ParsedTransaction[]): string
```
Format each transaction as `${t.date} | ${t.description} | €${Math.abs(t.amountEur).toFixed(2)}`, joining with newlines. Prompt instructs the model to categorize into 4 buckets (Everyday Essentials: groceries, utilities, transport, healthcare; Fun & Discretionary: restaurants, entertainment, subscriptions, hobbies, shopping; One-off & Travel: holidays, flights, hotels, large one-time purchases; Recurring Fixed: regular subscriptions, insurance, gym, phone plans), calculate average monthly spending, explain reasoning, suggest monthly budget rounded to nearest €10.

**Main function:**
```typescript
export async function callAnthropicAPI(
  apiKey: string,
  transactions: ParsedTransaction[]
): Promise<AIAnalysisResult>
```
- POST to `https://api.anthropic.com/v1/messages`
- Headers: `x-api-key`, `anthropic-version: 2023-06-01`, `content-type: application/json`, `anthropic-dangerous-direct-browser-access: true`
- Body: `{ model: 'claude-haiku-4-5-20251001', max_tokens: 2048, messages: [{role:'user', content: buildAnalysisPrompt(transactions)}], output_config: { format: { type: 'json_schema', schema: ANALYSIS_SCHEMA } } }`
- On non-OK response: read JSON body, extract `error.type` (default 'unknown_error'), throw `new AnthropicAPIError(response.status, errorType)` — NEVER include apiKey in error message
- On OK: return `JSON.parse(data.content[0].text) as AIAnalysisResult`

CRITICAL: Do NOT import from React. This module must be importable in Vitest/Node.
CRITICAL: Never log or include apiKey in any error message, thrown string, or console output.

**Analyse handler:**
```typescript
const handleAnalyse = async () => {
  const key = localStorage.getItem('anthropic_api_key');
  if (!key || transactions.length === 0) return;
  setIsAnalysing(true);
  setAnalysisError(null);
  try {
    const result = await callAnthropicAPI(key, transactions);
    setAnalysisResult(result);
  } catch (e) {
    if (e instanceof AnthropicAPIError) {
      setAnalysisError(`API error ${e.status}: ${e.errorType}`);
    } else {
      setAnalysisError('Network error — check your connection and retry');
    }
  } finally {
    setIsAnalysing(false);
  }
};
```
