The complete updated `applyTheme`:
```typescript
function applyTheme(theme: 'light' | 'dark' | 'system'): void {
  const root = document.documentElement;
  const prefersDark =
    typeof window !== 'undefined' &&
    window.matchMedia != null &&
    window.matchMedia('(prefers-color-scheme: dark)').matches;
  const isDark = theme === 'dark' || (theme === 'system' && prefersDark);
  root.classList.toggle('dark', isDark);
  try { localStorage.setItem('mf_theme', theme); } catch (_) {}
}
```
This is the FOUC mirror — every theme change (both from toggle and from loadSettings) writes to localStorage so the inline script can read it on next page load.
