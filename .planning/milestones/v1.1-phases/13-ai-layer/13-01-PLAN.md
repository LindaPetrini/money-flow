**New types to add (export all):**

```typescript
export interface UncertainTransaction {
  description: string;   // original CSV description (exact case — used for merchant lookup)
  date: string;          // ISO YYYY-MM-DD
  amountEur: number;     // negative for expenses
  reason: string;        // why AI is uncertain — shown in Q&A card
}

export interface CombinedAnalysisResult {
  uncertainTransactions: UncertainTransaction[];
  everydayEssentials: BucketSuggestion;
  funDiscretionary: BucketSuggestion;
  oneOffTravel: BucketSuggestion;
  recurringFixed: BucketSuggestion;
}

export interface FloorItemSuggestion {
  name: string;        // suggested floor item name e.g. "Rent"
  amountEur: number;   // detected recurring amount — positive, AI returns absolute value
  frequency: string;   // "monthly" | "quarterly" — for transparency display
  confidence: string;  // "high" | "medium" — for display
  reason: string;      // why this looks like a floor item
}

export interface FloorDetectionResult {
  suggestions: FloorItemSuggestion[];
}
```

**New schemas (module-level, not exported — internal only):**

UNCERTAIN_TRANSACTION_SCHEMA: object with description/date/amountEur/reason, all required, additionalProperties: false

COMBINED_ANALYSIS_SCHEMA: object with uncertainTransactions (array of UNCERTAIN_TRANSACTION_SCHEMA), everydayEssentials/funDiscretionary/oneOffTravel/recurringFixed (all BUCKET_SCHEMA), all required, additionalProperties: false

FLOOR_ITEM_SUGGESTION_SCHEMA: object with name/amountEur/frequency/confidence/reason, all required, additionalProperties: false

FLOOR_DETECTION_SCHEMA: object with suggestions (array of FLOOR_ITEM_SUGGESTION_SCHEMA), required: ['suggestions'], additionalProperties: false

**Prompt builders (module-level, not exported):**

`buildCombinedAnalysisPrompt(transactions: ParsedTransaction[]): string`
- Format each transaction as: `${t.date} | ${t.description} | €${Math.abs(t.amountEur).toFixed(2)}`
- Instructions: classify into 4 buckets + identify uncertain transactions
- Specify: amountEur is always a negative number (expense). Do NOT use minimum/maximum in schema.
- Ask AI to flag as uncertain: unknown merchants, first-time payees, ambiguous descriptions
- Response includes both bucket analysis AND uncertainTransactions list

`buildFloorDetectionPrompt(transactions: ParsedTransaction[], clarifications: Array<{merchantName: string; bucketAccountId: string; context?: string}>): string`
- List all transactions
- List clarifications (already answered — skip these as floor candidates)
- Ask: which transactions appear to be recurring monthly/quarterly fixed expenses?
- amountEur in response is always a positive number representing the expense magnitude
